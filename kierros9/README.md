# Tehtävä 6d:

Brute force -lähestymistapa (esim. jakajien etsiminen):
Plussat: erittäin yksinkertainen hahmottaa.
Miinukset: selvästi hitain vaihtoehto suurilla luvuilla.

Rekursiivinen ratkaisu:
Plussat: koodi on usein lyhyt ja ilmaisultaan hyvin lähellä matemaattista määritelmää.
Miinukset: käyttää kutsupinoa, voi olla hitaampi, ja syvä rekursio voi aiheuttaa pinon ylivuodon.

Iteratiivinen (silmukkaperusteinen) ratkaisu:
Plussat: toimii nopeasti, vie vähän muistia eikä aiheuta kutsupinoon liittyviä ongelmia.
Miinukset: voi tuntua vähemmän “matemaattisesti elegantilta” verrattuna rekursioon.
